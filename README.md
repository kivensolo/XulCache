# What is XulCache ?
The cache module extracted from the [starcor-xul](https://github.com/starcor-company/starcor.xul) framework.

# How to use?
## project's build.gradle file:
```
 buildscript {
    repositories {
        ...
        jcenter()
        ...
    }
 }
```

## app's build.gradle file:
```
dependencies {
  implementation 'com.github.xul:cache:1.0.0'
}
```
